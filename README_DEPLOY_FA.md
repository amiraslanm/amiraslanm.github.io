# نسخه حرفه‌ای سایت Amir Aslan Mosaferi

## روش جایگزینی سایت فعلی در GitHub
1. فایل ZIP را Extract کنید.
2. وارد Repository با نام `amiraslanm.github.io` شوید.
3. از `Add file` گزینه `Upload files` را انتخاب کنید.
4. **تمام محتویات داخل پوشه** را Drag & Drop کنید؛ خود پوشه یا ZIP را آپلود نکنید.
5. GitHub فایل‌های هم‌نام را به‌روزرسانی می‌کند. پیام Commit را `Upgrade portfolio to premium v2` بگذارید و Commit کنید.
6. چند دقیقه بعد سایت `https://amiraslanm.github.io` خودکار به‌روزرسانی می‌شود.

## شخصی‌سازی مهم بعدی
- عکس پرتره حرفه‌ای شما در نسخه فعلی عمداً با گرافیک AM جایگزین شده است.
- لینک LinkedIn هنوز اضافه نشده چون URL قطعی در اختیار نبود.
- برای دامنه شخصی، بعداً می‌توان `amirmosaferi.com` را متصل کرد.

## ساختار
- `index.html`: صفحه اصلی
- `projects/`: مطالعات موردی
- `posts/`: یادداشت‌های فنی
- `blog.html`: آرشیو مطالب
- `cv.html`: رزومه قابل چاپ و ذخیره PDF
- `assets/`: CSS، JavaScript و تصاویر
