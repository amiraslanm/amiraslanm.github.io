# Default English Website Update

This update changes the website language strategy:

## Main behavior
- The default website language is now English.
- Pages load as `lang="en"` and `dir="ltr"`.
- The language toggle button shows `فا` on first load.
- Visitors can switch to Persian manually.
- If a visitor chooses Persian, the preference is saved in their browser only.

## Why
This is better for international PhD supervisors, research collaborators, employers, and academic reviewers.
