# نسخه محرمانگی‌محور پورتفولیو

اصول انتشار پروژه‌ها:
- فقط نقش، روش حل، استانداردها، نرم‌افزارها و فرآیند تصمیم‌گیری نمایش داده می‌شود.
- نام کارفرما، موقعیت، کد ساختمان، شناسه شبکه، مقادیر دقیق، نقشه‌ها و اطلاعات قراردادی منتشر نمی‌شوند.
- صفحات پروژه با نام‌های عمومی و غیرقابل ردیابی ارائه شده‌اند.
