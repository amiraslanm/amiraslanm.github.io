آپدیت تفکیک آموزش‌های کاری و پژوهشی

این بسته:
- صفحه education.html را به Learning Hub دوشاخه تبدیل می‌کند.
- آموزش‌های کاری را در Professional Learning قرار می‌دهد.
- آموزش‌های پژوهشی و CFD را در Academic Learning قرار می‌دهد.
- داخل research.html دکمه مستقیم به /cfd-learning/ اضافه می‌کند.
- داخل professional.html دکمه دسترسی به آموزش‌های کاری اضافه می‌کند.

مسیرهای مهم:
- education.html
- research.html
- professional.html
- index.html
- cfd-learning.html
- assets/learning-split.css
- sitemap.xml

روش آپلود:
1) ZIP را Extract کن.
2) در GitHub Repository روی Add file > Upload files بزن.
3) همه فایل‌های داخل پوشه را آپلود کن.
4) Commit message:
   Split learning hub into professional and academic tracks
5) Commit directly to main
6) بعد از چند دقیقه Ctrl + Shift + R بزن.
