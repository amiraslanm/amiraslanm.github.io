آپدیت OpenFOAM برای سایت GitHub

روش آپلود:
1) فایل ZIP را Extract کن.
2) داخل Repository سایت برو.
3) روی Add file > Upload files بزن.
4) همه فایل‌ها و پوشه‌های داخل فولدر Extract شده را Drag & Drop کن.
5) مطمئن شو این مسیرها دیده می‌شوند:

openfoam-training.html
certifications.html
education.html
research.html
cv.html
index.html
sitemap.xml
assets/openfoam-training.css
assets/certificates/basic-openfoam-certificate-privacy.webp
assets/certificates/basic-openfoam-certificate-privacy.jpg
README_OPENFOAM_TRAINING_FA.md

6) Commit message:
Add OpenFOAM training notes and certificate

7) Commit directly to main
8) بعد از چند دقیقه Ctrl + Shift + R بزن.
