بسته آپدیت دوره پوشش‌های مقاوم حریق

1) فایل ZIP را Extract کنید.
2) در GitHub Repository سایت، Add file > Upload files را انتخاب کنید.
3) همه فایل‌ها و پوشه‌های داخل بسته را آپلود کنید.
4) مسیرهای مهم:
   fire-resistant-coatings.html
   education.html
   index.html
   assets/education.css
   assets/education.js
   sitemap.xml
5) Commit message:
   Add fire-resistant coatings education course
6) Commit directly to main
7) چند دقیقه بعد سایت را با Ctrl + Shift + R بازخوانی کنید.
