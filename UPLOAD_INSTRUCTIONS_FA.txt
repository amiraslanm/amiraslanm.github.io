آپدیت مبنای انگلیسی سایت

این آپدیت باعث می‌شود سایت برای همه بازدیدکننده‌های جدید اول انگلیسی باز شود.
فارسی فقط با دکمه «فا» فعال می‌شود.

روش آپلود:
1) ZIP را Extract کن.
2) داخل Repository اصلی سایت برو:
   amiraslanm/amiraslanm.github.io
3) روی Add file > Upload files بزن.
4) همه فایل‌ها و پوشه‌های داخل فولدر Extract شده را Drag & Drop کن.
5) Commit message:
   Set English as default website language
6) Commit directly to main
7) بعد از Deploy سایت را با Ctrl + Shift + R رفرش کن.

نکته:
اگر مرورگر خودت قبلاً فارسی را ذخیره کرده باشد، ممکن است برای خودت فارسی باز شود.
برای تست مثل استادها، سایت را در Incognito باز کن:
https://amiraslanm.github.io/
