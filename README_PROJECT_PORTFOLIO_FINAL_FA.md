# نسخه نهایی صفحه Engineering Projects

این پکیج متن‌های صفحه `engineering-projects.html` را حرفه‌ای‌تر و مناسب‌تر برای LinkedIn، رزومه و اپلای دکتری می‌کند.

## تغییرات اصلی
- حذف ادبیات آرشیوی مثل `first archive period` و `second archive period`
- جایگزینی با عبارت‌های حرفه‌ای‌تر مثل `fire protection and mechanical safety records summarized`
- تقویت متن معرفی صفحه پروژه‌ها
- حرفه‌ای‌تر شدن دسته‌بندی خدمات مهندسی
- تقویت یادداشت محرمانگی

## روش آپلود
محتویات فایل ZIP را Extract کن و فایل‌های داخل پوشه را در root ریپازیتوری GitHub Pages آپلود کن؛ یعنی همان جایی که `index.html` قرار دارد.
