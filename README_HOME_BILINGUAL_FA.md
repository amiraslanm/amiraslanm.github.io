# Home Page Bilingual Fix

این آپدیت صفحه اصلی سایت را کاملاً دوزبانه می‌کند.

تغییرات:
- زبان پیش‌فرض صفحه اصلی فارسی شد.
- تمام متن‌های اصلی صفحه index.html دارای data-fa و data-en شدند.
- با کلیک روی دکمه EN، متن‌های صفحه اصلی به انگلیسی تبدیل می‌شوند.
- با کلیک روی دکمه فا، متن‌ها دوباره فارسی می‌شوند.
- عنوان تب مرورگر هم بین فارسی و انگلیسی تغییر می‌کند.

فایل‌های تغییرکرده:
- index.html
- assets/app.js
